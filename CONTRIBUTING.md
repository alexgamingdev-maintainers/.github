# Contributing to Alex Gaming Studio

First of all, thank you for showing interest in improving our projects! We appreciate the community's effort to make our templates and tools better.

## üìú Our Policy

Before you start, please understand our core rules regarding our codebase:

1.  **Pull Requests:** We openly welcome Pull Requests! If you find a bug, want to optimize code, or add a feature to a template, feel free to contribute.
2.  **No Cloning/Copying:** Cloning, forking, or copying our projects and code for personal use, re-distribution, or commercial purposes is **strictly prohibited**.
3.  **Ownership:** By submitting a Pull Request, you agree that your contributions will be part of the project under its existing restrictions.

## üõ†Ô∏è How to Contribute

### 1. Reporting Bugs
If you find a technical issue in one of our tools:
* Check if the issue has already been reported.
* If not, open a new issue and describe the bug, including steps to reproduce it.

### 2. Suggesting Enhancements
Have an idea for a new feature in a template?
* Open an issue to discuss the idea first.
* This ensures that the enhancement fits the vision of the project.

### 3. Submitting a Pull Request
* Ensure your code follows the existing style of the project.
* Provide a clear description of what your PR changes or fixes.
* Once submitted, we will review your code as soon as possible.
